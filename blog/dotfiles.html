learning to appreciate dotfiles

march 24 2026

for years i've had an incredibly undermaintained dotfiles repo that did nothing. i'd always used cloud sync or just manually recreated configs.

this was fine until i switched to zed, and started sending myself the config file in signal's note to self feature. this is very very scuffed, but it worked out because it was the only app i used like this.

then i got frustrated with warp and switched to ghostty, meaning now i had another config file floating around, once again, fine because i can just send it to myself for the two apps that use it.

now we're to now, after leaving warp, i had to start recreating functionalities i used with zshrc and ohmyzsh, which is where things start to get painful.

every time i update my zshrc, i have to remember to send it to myself before closing the file. this fucking sucks. and now that i'm preparing to switch to atomic, and started using flox, i have yet another thing i can't set up graphically/with a package.

my philosophy has always been to try to use the gui for as much as possible, it's convenient and i get to find papercuts before my ultramarine users do. despite this i am rather fluent in the terminal (except with git). this philosophy isn't changing at large, but as i use more and more software that's built by devs for devs, i'm forced to re-enter this world of configs.

so now i have a dotfiles repo again, and it actually exists this time. it contains my fancy zshrc, my ghostty configs for mac and linux, my zed config, and a script that installs all of them.

i'd suggest not using the script or zshrc if you're not me (it sets up my git credentials and makes assumptions, like my flox envs), but feel free to try those configs, i think they're pretty good!

more stuff will be landing here in the future, i may stop using floxhub for my default environments, jadeite may move here (not sure yet) and i wanna throw a niri+DMS config in there for when i'm feeling like a window manager.

anyway that was really rambly, thanks for sticking with it, check out my dotfiles here. and be ready for a post about getting fyra repos up on devcontainers (and flox sometimes) soon!🐾
